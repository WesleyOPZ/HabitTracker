# 🎯 Habit Tracker

A console application to build and track daily habits with gamification elements.

## Features

- ✅ Create and manage habits
- 🔥 Track daily streaks
- ⭐ Earn XP for completing habits
- 📊 View statistics and history
- 💾 Persistent storage (JSON)

## Technologies

- C# / .NET 8
- JSON serialization
- Console interface

## How to run
```bash
dotnet run
```
